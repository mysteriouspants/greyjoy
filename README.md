# Greyjoy Octopress Theme

A simple white and grey colored theme for Octopress. Named after the messed up Theon Greyjoy.

## Demo

See http://reza.re

## Install

```
$ cd .themes
$ git clone git://github.com/rezajatnika/greyjoy.git
$ rake install[greyjoy]
$ rake generate
```
